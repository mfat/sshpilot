# Nutzungstipps für SSH Pilot
#
# Ein Tipp pro Zeile. Leere Zeilen und Zeilen, die mit "#" beginnen, werden ignoriert.
# Diese Datei ist die deutsche Fassung von tips.md — Tipps können hier bearbeitet,
# ergänzt oder entfernt werden, ohne den Quelltext zu ändern.

Wählen Sie einen Host aus und drücken Sie Strg+Umschalt+O, um den Dateimanager zu öffnen!
Drücken Sie Strg+F, um Ihre Verbindungen zu durchsuchen
Drücken Sie Strg+Umschalt+N, um eine neue Verbindung zu öffnen
Drücken Sie Strg+Umschalt+F, um im Terminal zu suchen
Drücken Sie Strg+Umschalt+K, um Ihren SSH-Schlüssel auf einen Server zu kopieren
Drücken Sie F9, um die Seitenleiste ein- oder auszublenden
Drücken Sie Strg+?, um alle Tastenkürzel anzuzeigen
Klicken Sie mit der mittleren Maustaste auf eine Verbindung, um sie in einem neuen Tab zu öffnen!
Klicken Sie mit der mittleren Maustaste auf einen Tab, um ihn zu schließen!
Zeigen Sie mit der Maus auf die rechte Seite einer Verbindung, um den Dateimanager zu erreichen
In den Einstellungen können Sie Ihr bevorzugtes Terminal auswählen
Tippen Sie nach dem Start einfach einen Hostnamen ein und drücken Sie die Eingabetaste, um sich zu verbinden!
